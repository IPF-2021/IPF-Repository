## Instalation

- Clone the repository

  ```
    git clone https://www.github.com/IPF-2021/IPF-Repository
  ```

- Install the dependencies

  ```
    npm install
  ```

## Inicialization

### Start the project in production mode
-
  ```
    npm run start
  ```
  or
-
  ```
    npm start
  ```

### Start the project in develpment mode
-
  ```
    npm run dev
  ```
